# Jharkhand Landscape — Who Does What Where

An interactive, fully self-contained (offline, no keys/CDN) sense-making view of the
development-partner ecosystem in Jharkhand: **partners × districts × themes**, with TRI
presence, Common Ground blocks and CSR flow — plus **Ecosystem Health** and **Place Health**
scorecards.

**Live:** https://ashwask.github.io/jharkhand-landscape/

## What's inside `index.html`
- District choropleth with 5 lenses (partner density, theme breadth, CSR spend, dominant theme, coverage gap)
- Click-a-district detail: partners, themes, TRI/Common Ground blocks, 10-year CSR trend
- **Ecosystem Health index** (coverage, aspirational reach, resilience, thematic balance, network depth, resource alignment)
- **Place Health** — every district scored 0–100, ranked neediest-first
- Partner × Theme matrix, sortable partner directory and district coverage table

## Data
Built from partner geography/thematic sheets, TRI Geographic Presence (Jul 2026), the Common
Ground block list, and MCA district-wise CSR data. District boundaries © [udit-001/india-maps-data](https://github.com/udit-001/india-maps-data) (2011 census).
District names and themes are normalised; some partner rows list out-of-state / town names and are kept off the map.

*Internal sense-making tool. District labels ("whitespace", "priority") describe partner-coverage gaps, not judgements of the districts or partners.*
